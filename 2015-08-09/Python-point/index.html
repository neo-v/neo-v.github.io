Python编程要点拾遗。

  1. 参数传递
    做leetcode发现，Python在参数传递，是传值还是传 地址的不同，不同于c/c++有指针，Python的传递看起来不是很直接。
    翻了资料发现，Python数据类型是分可变和不可变的。对于不可变的数据类型，比如整数，是copy的。对于可变的，比如list，是传地址的，或者说引用吧。所以说兼具了传值和传地址。在使用的时候，就需要注意。
    如果需要传递copy，可以对list做切片：

    copy = list[:]

    如上。

  2. 迭代器

    迭代器，也就是一般来说的循环。Python里有语法：

    for item in range(m):
        ...

    其他编程语言必支持的流程控制语句。Python for语句对循环的容器，实际调用内建的iter()方法，获得迭代器，然后在调用obj.next()方法，遍历容器的元素。Python并没有类似Java，has_next()方法。利用的是异常。

  3. yield

    函数带有yield语句，在执行的时候，保存当前状态，并返回到调用的地方。直到下次继续调用该方法的时候，继续从yield下一条指令开始执行。这是个很有用的特性。用来实现生成器。

  4. with

    with语句，相当于try…catch语句的一种代替。有点类似c++的RAII技术。典型的例子：

    with context_expression [as target(s)]:
    with-body

    这里，with生成了上下文管理器对象，在with-body运行前，运行管理器对象的_enter()方法，在结束运行\exit()方法。这里context_expression要返回一个上下文管理器对象，该对象并不赋值给 as 子句中的 target(s) ，如果指定了 as子句的话，会将上下文管理器的\enter_() 方法的返回值赋值给 target(s)。
    举个例子：

    with open(r'somefileName') as somefile:
    for line in somefile:
        print line
        # ...more code

    结束时，自动释放file资源。

  5. 生成器
    生成器可以做，迭代器的全部事情。利用的就是yield语句。包含了yield语句的函数，就是一个生成器。使用生成器最大的好处，是不必一次缓存所有数据，而在每一次迭代中动态生成。如果迭代的数据量很大，或者内存敏感，就应该使用生成器。
    举个例子：

    def reverse(data):
    for index in range(len(data)-1,-1, -1):
        yield data[index]

for char in reverse('hello'):
    print(char)

    这里print（char)的时候数据是在yield语句迭代生成的。实现上，生成器函数自动生成了iter()和 next()方法。

  6. xrange()
    xrange() 与range()的不同，就在与它是个生成器。

  7. 装饰器
    包装函数，将目标函数包裹起来，做一些其他事情，对外可见的是包装后的函数。比如要对函数计时。

    def timethis(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(func.__name__, end-start)
        return result
    return wrapper

@timethis
def countdown(n):
    while n > 0:
        n -= 1

countdown(100000)

    @timethis 语法糖，标志了该函数的包装。在countdown调用时候，实际调用是，timethis(countdown)返回的wrap函数。

  8. 列表解析
    使用列表解析可以简化代码的书写。而且避免不必要的编程逻辑。
    举个例子，从一个输入列表中将所有大于0的整数平方生成一个新的序列，不使用列表解析：

    num = [1, 4, -5, 10, -7, 2, 3, -1]
filtered_and_squared = []
for number in num:
    if number > 0:
        filtered_and_squared.append(number ** 2)
print filtered_and_squared

    需要初始化空列表，然后三行代码。使用列表解析：

    num = [1, 4, -5, 10, -7, 2, 3, -1]
filtered_and_squared = [ x**2 for x in num if x > 0]
print filtered_and_squared

    可以看到非常简洁。而且省去了初始化空列表。重要的一点，它使你看起来是Python coder。。

  10. 命名空间和作用域
    Python 有四个作用域，分别为：
    local，enclose，global，built_in。符号解析的时候，依次搜索。找不到会抛出NameError异常，只有模块，类，和函数会引入新的作用域。
    作用域是可以嵌套的，内层赋值语句会引入新的变量，有可能覆盖全局的变量。所以不推荐：

    from a_module import *
    导入变量有可能被覆盖。
